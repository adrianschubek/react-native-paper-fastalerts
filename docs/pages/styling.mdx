# Styling

Even though the default styling should be sufficient for most use cases, you can customize the look and feel of the alert through different options.

### Title

```tsx {3-7}
alert({
  title: "Hello World!",
  titleStyle: {
    color: "red",
    fontSize: 24,
    fontWeight: "bold",
  }
});
```

### Message

```tsx {3-7}
alert({
  message: "Hello World!",
  messageStyle: {
    color: "red",
    fontSize: 24,
    fontWeight: "bold",
  }
});
```
